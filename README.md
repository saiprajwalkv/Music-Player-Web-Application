# Music-Player-Web-Application

A simple browser-based music player built using HTML, CSS, and JavaScript.  
The application allows users to play, pause, and navigate between songs.

## Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## Features
- Play and pause music
- Next and previous song navigation
- Seek bar for song progress
- Volume control
- Responsive user interface

## Note
UI design is inspired by popular music players and built for learning purposes.
